# 记账本

这是一个基于 React 18 + Vite 构建的个人记账 Web 应用，前端采用 Tailwind CSS 毛玻璃风格（Glassmorphism），后端使用 Supabase 托管 PostgreSQL 数据库和邮箱认证。用户注册登录后，通过抽屉面板记录收入/支出（含金额、12 个 emoji
  分类、日期和备注），首页仪表盘实时展示本月收入/支出/结余三张汇总卡片、Recharts 支出分类圆环饼图和最近 5 条账单列表（支持编辑删除），右侧箭头可切换历史月份查看过往数据，统计页提供近 6
  个月收支对比柱状图和分类排行进度条。项目已部署至 GitHub Pages，数据库运行在 Supabase 免费层，整套零成本。
